ITOSS Manager Basic Administration

This section contains information on the basic administration activities of the ITOSS software platform. This software component is the center of the platform, and any unavailability affects the entire service, comprising the back-end and front-end modules.

Directories, Logs, and Statisticsâ€‹

Base directory:

/app/itoss/itoss-manager/

Binary directory:

/app/itoss/bin

Graphic Interface directory:

/app/itoss/www/itoss

Help Desk Integration directory:

/app/itoss/integration

Log file:

/app/itoss/itoss-manager/logs/itoss-manager.log

Configuration file:

/app/itoss/itoss-manager/config/application.properties

Statistics:

curl http://<manager_endpoint>/stats |python -m json.tool

Status, Start, and Stopâ€‹

ITOSS Manager Status:

systemctl status itoss-manager

Start ITOSS Manager:

systemctl start itoss-manager

Stop ITOSS Manager:

systemctl stop itoss-manager

Web Service (UI) Status:

systemctl status nginx

Start Web Service (UI):

systemctl start nginx

Stop Web Service (UI):

systemctl stop nginx

Web Service Configuration file:

/etc/nginx/conf.d/default.conf
