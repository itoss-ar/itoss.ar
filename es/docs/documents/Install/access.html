Access to the user interface

The ITOSS user interface is accessible via a modern web browser compatible with JavaScript. Upon entry, access credentials will be requested. From a web browser, enter the URL of the ITOSS web server and enter the user credentials. http://localhost/itoss/

Note: ITOSS has been extensively tested with Google Chrome, and we recommend using this browser.
